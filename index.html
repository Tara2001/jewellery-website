Overview

One website template is an creative, clean and fully responsive template, built on 1170px Grid System of Twitter Bootstrap 3. Template comes with developer friendly and easy to customize code. Oneis compatible with all major web browsers, tablets and mobile devices.

We hope you will enjoy using our template. it's easy to navigate, user-friendly alongwith developer friendly code for any kind of customization or changes.
